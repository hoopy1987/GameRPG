# 炭火村 NPC 角色设定与职业体系

> 文档版本：v1.0
> 适用场景：游戏文案基准、对话编写、任务设计、角色精灵图选型

---

## 一、核心原则

1. **正式姓名**：每名 NPC 拥有完整姓名，禁用"老张"之类外号
2. **功能绑定**：每个 NPC 必须有明确游戏功能（卖东西 / 给任务 / 提供情报 / 训练技能）
3. **对话风格**：与其身份、性格、当前处境匹配
4. **背景自洽**：所有 NPC 的存在理由必须在村庄生态中说得通

## 三、角色详细设定

### 1. 格雷恩·铁锤（Graen Ironhammer）— 铁匠

**职业**：铁匠 / 武器锻造师
**年龄**：48 岁
**外貌**：红脸膛、络腮胡、右臂比左臂粗一圈，围裙上永远有火星烧出的洞

**身份背景**：
炭火村唯一的铁匠，祖上三代在此打铁。矿洞出事前，他的铁匠铺日夜不熄；现在矿石来源断了，他只能靠库存和收购冒险者带来的矿石维持。他的妻子三年前死于矿洞塌方，从此他对"矿洞深处"这个词有复杂情绪——既恨那里吞噬了他的妻子，又依赖那里的矿石维持生计。

**游戏功能**：
- 出售：铁剑、钢剑、盾牌、头盔（基础装备）
- 锻造升级：玩家提供矿石 + 金币，可将武器升级（+攻击力 / +暴击率）
- 修理：装备耐久度修复
- 收购：收购玩家从矿洞带回的矿石

**对话风格**：
- 豪爽、直接、不爱绕弯子
- 提到妻子时会突然沉默，然后转移话题
- 对矿洞话题敏感："那鬼地方……算了，你要买剑还是修盾？"

**示例对话**：
```
【初次见面】
"新来的？看你这身装备，连只地精都打不过。
我是格雷恩·铁锤，这村子唯一的铁匠。
需要武器？找我。需要建议？也找我——但我只懂怎么把东西打得更硬。"

【购买武器】
"这把铁剑？五个银币，不还价。
……好吧，四个半。但别告诉别人我给了你折扣，毁我招牌。"

【提及矿洞】
"矿洞？（停顿，锤子停在半空）
那地方……吃人不吐骨头。我妻子就——
（突然用力敲了一下铁砧，火星四溅）
……你还要买什么？"

【升级武器】
"想淬火？拿来！
……矿石品质不错。三个金币，我给你加道血槽，砍东西更顺手。
不，血槽不是真的血，你这读书人……"
```

### 2. 玛格丽特·麦酒（Margaret Mead）— 酒馆老板娘

**职业**：酒馆老板娘 / 情报贩子
**年龄**：52 岁
**外貌**：微胖、银发盘成发髻、永远系着沾有酒渍的围裙、眼神锐利如鹰

**身份背景**：
炭火村"熔炉与麦穗"酒馆的老板娘，也是全村消息最灵通的人。三十年前和丈夫一起开了这家店，丈夫五年前病死后她独自经营。她的酒馆不只是喝酒的地方——冒险者、商人、甚至路过的斥候都会在这里交换情报。她卖情报和卖麦酒一样熟练。

**游戏功能**：
- 出售：麦酒（小幅 HP 恢复）、炖肉（中幅 HP 恢复）、面包
- 情报交易：花金币购买关于矿洞、遗迹、怪物动向的情报
- 休息存档：在酒馆住宿一晚 = 存档 + 完全恢复 HP
- 支线任务发布：找人、送物、打听消息

**对话风格**：
- 圆滑、八面玲珑、从不把话说死
- 擅长从对话中套出信息，再包装成"情报"卖给别人
- 对金币敏感，但也会因"投缘"给出折扣

**示例对话**：
```
【初次见面】
"欢迎光临'熔炉与麦穗'，我是玛格丽特。
你是从东边来的？北边？……无所谓，只要你会付账，从地狱来我也招待。
要喝点什么？麦酒三个铜板，情报……看你想知道什么。"

【购买情报】
"矿洞第二层的情报？五个金币。
（压低声音）
最近有猎人看到蝙蝠群比往常多三倍……还有人在深处听到了金属撞击声。
不是挖矿声。是某种……机械。
我只说这么多。想要更多？加钱。"

【休息存档】
"要住一晚？一个金币，包晚饭和早餐。
……两个金币可以住阁楼，安静，没人打扰。
（眨眼）
阁楼能看到星星，适合……整理思绪。"

【闲聊】
"村长埃尔温？好人，就是太操心。
格雷恩？嘴硬心软，他工坊里还留着他妻子的织布机，舍不得扔。
至于矿洞……（声音更低）
有人说在深处看到了光。不是火把光。是蓝色的。像……眼睛。"
```

### 3. 艾德里安·圣烛（Adrian Holywax）— 牧师

**职业**：牧师 / 治疗者
**年龄**：61 岁
**外貌**：瘦高、白发、永远披着褪色的蓝色圣袍、手持一根顶端嵌有微光水晶的法杖

**身份背景**：
炭火村教堂的唯一神职人员。三十年前从教廷派遣至此，原本只打算待三年，却因爱上了这个村子的宁静而留下。他见证了村庄从繁荣到矿难后的萧条。他相信矿洞的灾难不是偶然，而是某种"古老的警告"。他的治疗能力来自虔诚祈祷，而非魔法——至少他自己是这么说的。

**游戏功能**：
- 治疗：完全恢复 HP，免费（但每天有次数限制）
- 祝福增益：花钱购买临时增益（+防御、+抗性）
- 宗教任务：净化被诅咒物品、寻找圣遗物、超度亡灵
- 复活：玩家死亡后可在此复活（比随机重生惩罚更小）

**对话风格**：
- 庄严、温和、带有布道口吻
- 引用圣典和谚语频繁
- 谈到矿洞深处时会流露真实恐惧

**示例对话**：
```
【初次见面】
"愿圣光照耀你，旅人。
我是艾德里安·圣烛，这座小教堂的看守者。
如果你受伤了，进来吧。治疗不收费——圣光从不索取回报。
但如果你有金币想捐赠……（微笑）
教堂的屋顶漏雨三个月了。"

【治疗】
"坐下，深呼吸。
（法杖轻点你的肩膀，微光流转）
'伤口会愈合，正如黑夜会过去。'
——《晨光圣典》第三章第七节。
好了。今天还能为你治疗两次。"

【购买祝福】
"你需要防御祝福？三个金币，持续一个时辰。
这不是魔法，孩子。是圣光借我的身体流入你。
……好吧，从效果上看和魔法差不多。但来源不同，这很重要。"

【谈及矿洞】
"矿洞？（声音变低，法杖握紧）
三十年前，我刚到这里时，矿工们在最深处挖到了……东西。
不是矿石。是墙。一面刻满符文的墙。
第二天塌方了。死了十七人。
我相信那是警告。不是意外。
（沉默片刻）
……去祈祷吧，孩子。你需要的不只是装备。"
```

### 4. 埃尔温·禾穗（Elwin Wheatfield）— 村长

**职业**：村长 / 前农民
**年龄**：67 岁
**外貌**：佝偻、皱纹深刻、拄着一根打磨光滑的橡木手杖、眼神疲惫但温暖

**身份背景**：
炭火村的村长，也是村庄最年长的居民。他年轻时是村里最好的农民，后来在矿洞繁荣时期被选为村长。矿难发生后，他承受着巨大的 guilt——是他批准了更深层的挖掘。他现在的唯一目标是保住村庄不被废弃，同时尽量不让更多人去送死。

**游戏功能**：
- 主线任务发布：调查矿洞、寻找失踪矿工、封印深处威胁
- 村庄历史讲解：提供背景故事和世界观信息
- 资源调配：完成他的任务后解锁村庄设施升级（如铁匠铺扩建）
- 紧急求助：当村庄遭遇怪物袭击时发起防御战

**对话风格**：
- 沉稳、慈祥、但疲惫
- 经常陷入回忆，然后被现实拉回
- 对玩家的成长感到欣慰，像看孙子一样

**示例对话**：
```
【初次见面】
"啊……又一位冒险者。
我是埃尔温·禾穗，这个村子的……看守者吧，村长只是个称呼。
你是来发财的？来找刺激的？还是来……（叹气）
来送死的？
别生气，孩子。我见过太多人走进矿洞，没几个走出来。
"

【主线任务：调查矿洞】
"矿洞第二层最近传来奇怪的声音。
不是蝙蝠。不是老鼠。
是……歌声。有人说听到女人在唱歌。
（颤抖）
我妻子三十年前死在塌方里。她……喜欢唱歌。
去查清楚，好吗？带上这个灯笼，深处的黑暗……不只是没有光。"

【村庄历史】
"五十年前，这里只有三户人家，种麦子、养猪。
然后发现了银矿。一夜之间，商人、矿工、投机者涌进来。
我们建了教堂、铁匠铺、酒馆……我以为好日子来了。
（停顿，望向矿洞方向）
然后塌方。十七人。银矿断了。人走了。
现在只剩下我们这些老人，和像你这样……不怕死的年轻人。"

【任务完成反馈】
"你回来了。活着。
（露出真心的笑容，皱纹舒展开）
比我想象的强。比我见过的很多人都强。
……给你报酬。还有这个——我年轻时的护身符。
别推辞。你比我更需要它。"
```

### 5. 莉安娜·药钵（Liana Mortar）— 草药师

**职业**：草药师 / 炼金术士学徒
**年龄**：34 岁
**外貌**：苍白皮肤、深色长发编成复杂发辫、总是穿着沾有草药渍的深色斗篷、手指有淡淡的绿色药渍

**身份背景**：
来自北方王国的流浪草药师，三年前流浪到炭火村时因治愈了一场瘟疫而被留下。她住在教堂旁边的小屋里，平时采集森林和矿洞入口附近的草药。她与艾德里安牧师关系微妙——牧师认为她的"药水"效果太好，怀疑里面掺杂了不该有的东西，但从未挑明。她确实在秘密研究矿洞深处的某种发光苔藓。

**游戏功能**：
- 出售：高级生命药水（比商人卖的更有效）、解毒剂、抗性药水
- 收购：草药、发光苔藓、怪物器官（蝙蝠翅膀、蜘蛛毒囊）
- 炼金任务：收集稀有材料合成特殊药剂
- 配方传授：完成任务后解锁药水自制配方

**对话风格**：
- 冷静、专业、用词精确
- 略带神秘，不主动解释药水原理
- 提到"实验"或"研究"时会闪烁其词

**示例对话**：
```
【初次见面】
"你身上有伤口的味道。还有……铁锈？和血混在一起。
我是莉安娜。不是医生，是草药师。
区别？医生治人。我治……问题。
说吧，什么问题？"

【购买药水】
"高级生命药水？八个金币。
成分？水、草药精华、一点……催化剂。
别问催化剂是什么。问了我也不会说。
（低声）
总之喝了能活。这就够了。"

【收购材料】
"蝙蝠翅膀？给我看看。
（仔细检查）
品质一般。三个铜板。
……但如果你能从矿洞深处带发光苔藓回来，我付十个银币。
那东西只在最黑的角落长。别问我为什么知道。"

【炼金任务】
"我在研究一种能驱散恐惧的药剂。
配方只差一味：地底蝙蝠的腺体分泌物。
听起来恶心？确实恶心。
去矿洞第二层，找那种眼睛发红的蝙蝠。
带回来。我会教你制作'勇气药水'——关键时刻能救你一命。"
```

### 6. 卡斯帕·灰羽（Caspar Ashfeather）— 猎人

**职业**：猎人 / 皮甲匠
**年龄**：41 岁
**外貌**：精瘦、短发、穿着自制的皮甲、背复合弓、脸上有一道从眉骨延伸到下巴的旧伤疤

**身份背景**：
炭火村唯一的猎人，住在北围栏附近的一间自建木屋里。他不吃村庄分配的粮食，全靠打猎自给自足。矿洞塌方后，森林里的野兽行为变得奇怪——更 aggressive、更……聪明。他认为这和矿洞深处的"某种变化"有关，但说不出具体是什么。他 sells 皮甲给冒险者，但只卖给"看起来不会送死"的人。

**游戏功能**：
- 出售：皮甲、轻甲、复合弓、箭矢
- 狩猎任务：清除特定区域的危险野兽
- 追踪教学：花钱学习"追踪脚印"技能（在地图上显示怪物位置）
- 特殊服务：为玩家制作定制皮甲（需完成任务解锁）

**对话风格**：
- 沉默寡言、务实、不信任陌生人
- 一旦认可玩家，会给出极其实用的野外生存建议
- 对"城市来的冒险者"有轻微鄙视

**示例对话**：
```
【初次见面】
"……（从阴影中走出，弓箭半举）
你是谁？从哪儿来？去矿洞？
（打量你三秒，放下弓）
卡斯帕。猎人。
不卖东西给傻瓜。你看起来……还行。
要什么？"

【购买装备】
"这件皮甲？十二个银币。
野猪皮做的，我亲手剥的、亲手缝的。
能挡爪子，挡不住剑。你遇到拿剑的……跑。
弓？十五个银币。射程三十步，再远靠运气。
……好吧，四十步。但别指望四十步外射中眼睛。"

【狩猎任务】
"北边的林子里有只狼。不是普通狼。
我追踪它三周了。它……在绕圈子。像在学习我的路线。
（皱眉）
帮我杀了它。二十银币。但我跟你一起去——这活儿一个人干不了。
如果你活下来了，我教你追踪。"

【被认可后】
"你……比我想象的强。
（难得露出笑容，伤疤扭曲）
来，坐。喝我的酒。我酿的，比玛格丽特那老婆娘卖的还烈。
告诉你一件事：别在满月进矿洞。
不是迷信。我亲眼看到过……东西。在月光照不到的深处。"
```

### 7. 托拜厄斯·秤杆（Tobias Scale）— 行商

**职业**：行商 / 杂货贩子
**年龄**：55 岁
**外貌**：永远穿着鲜艳的橙色外套、腰上挂着叮当作响的钱袋、笑眯眯的眼睛几乎被皱纹淹没

**身份背景**：
炭火村最熟悉的陌生人。他每两周来一次，拉着装满货物的驴车，卖些村里买不到的东西：南方的丝绸、东方的香料、北方的御寒毛皮。他也是玩家接触"外部世界"的窗口。他知道很多远方的消息，但真假参半——有时是故意的，有时他自己也分不清。

**游戏功能**：
- 出售：杂货（绳子、火把、地图、异域小饰品）
- 异地情报：其他村庄的状况、王国政治、通缉令
- 特殊订单：预付款后，下次来时带特定商品
- 收购：收购玩家获得的一切（价格偏低，但什么都收）

**对话风格**：
- 精明、热情、永远在推销
- 报价时眼睛不眨，砍价时表情丰富
- 散布的"情报"有真有假，需要玩家自行判断

**示例对话**：
```
【初次见面】
"托拜厄斯·秤杆，为你服务！
从东方到西方，从雪山到沙漠——只要你想买，我就能卖！
……好吧，不一定。但我会想办法。
你运气不错，我刚好进了批新货。
看看？不看不收钱，看了也不收钱，买了才收钱！"

【砍价】
"这捆绳子？五个银币。绝对不贵！
……三个？三个我连驴的草料钱都不够！
（拍大腿）
四个！少一个铜板我都不卖！
……好吧三个半。最后报价。我是说真的。
……三个？（长叹）你是属鹫的，专啄我这种老实商人。"

【情报】
"想知道远方的消息？花钱，或者拿东西换。
（压低声音）
王国军队在东部边境集结。不是演习。
还有……（表情变严肃）
有三个村庄，和炭火村一样有矿洞的，在过去半年里……消失了。
不是被遗弃。是消失。没人出来。没人进去。连鸟都不飞过去。
……这只是传闻。至少我希望只是传闻。"

【特殊订单】
"想要北方的御寒斗篷？现在没有。
但如果你先付一半定金，下次我来时给你带。
……别用那种眼神看我，我不是骗子！至少这次不是。
（塞给你一张皱巴巴的纸条）
拿着，收据。我托拜厄斯从不赖账——只是偶尔迟到。"
```

### 8. 布丽奇特·炉火（Bridget Hearthfire）— 农民

**职业**：农民 / 番茄种植者
**年龄**：29 岁
**外貌**：晒黑的皮肤、扎着两条麻花辫、永远穿着沾满泥土的粗布裙、笑起来露出两颗小虎牙

**身份背景**：
炭火村最年轻的农民，继承了父母在东边的一小块农田。她种植番茄、土豆和少量草药。矿洞繁荣时期她种的东西不愁卖；现在游客少了，她经常抱怨"天气"——其实抱怨的是生活。她是村庄"普通村民"的代表，没有特殊能力，但知道很多村里的小事。

**游戏功能**：
- 出售：番茄、土豆、小麦（低价食材）
- 日常闲聊：提供村庄八卦、天气影响、作物状态
- 支线任务：帮忙送货、驱赶偷菜的野兽、找丢失的工具
- 隐藏剧情：完成一系列帮她忙的任务后，她会送你一颗"祖传番茄种子"（可种植出恢复 HP 的特殊番茄）

**对话风格**：
- 朴实、乐观、爱抱怨天气（无论好坏）
- 闲聊时停不下来，说到正事反而结巴
- 对矿洞话题回避——她的哥哥死于塌方

**示例对话**：
```
【初次见面】
"哟！新面孔！
我是布丽奇特，种番茄的。
你想买番茄？一铜板一个，甜得像蜜。
……不想买？也行，聊聊天呗！
村里好久没新人了，我都快和番茄说话了。"

【购买食材】
"番茄一铜板，土豆两铜板，小麦……三个铜板一把。
（突然压低声音）
其实小麦不该这么贵，但格雷恩非要吃白面包，我只能涨价。
别告诉他是我说的！
（大声）
总之！新鲜！好吃！买吧！"

【天气抱怨】
"今天太阳太毒了，番茄都快晒成干了。
……昨天下雨又说太潮。
（耸肩）
反正天气永远是错的。人也是。算了，至少番茄不会骗我。"

【提及哥哥】
"矿洞？（笑容僵住，转过身去整理篮子）
……我哥哥在里面工作。塌方的……十七人之一。
（声音变轻）
他答应过要帮我修篱笆。篱笆还在。他……不在了。
（突然转身，强笑）
……你还要番茄吗？今天特价，两个一铜板。"
```

### 9. 汤姆·石墙（Tom Stonewall）— 守卫

**职业**：守卫 / 前矿工
**年龄**：38 岁
**外貌**：魁梧、寸头、左耳缺了一小块（矿洞事故）、穿着自制的简陋锁子甲、手持一根削尖的木棍当长矛

**身份背景**：
炭火村唯一的职业守卫。他原本是矿工，塌方中幸存但左耳受伤，从此不敢再下矿洞，转而当了守卫。他的"装备"很寒酸——锁子甲是格雷恩用废铁片拼的，长矛是削尖的木棍。但他非常忠诚，视村庄为生命。最近他注意到森林和矿洞方向的怪物活动越来越频繁，暗中加强了巡逻。

**游戏功能**：
- 出入盘查：第一次进入村庄时会询问来意
- 怪物预警：当村庄附近怪物密度过高时发出警告
- 防御任务：村庄遭遇袭击时加入战斗，玩家可协助防守
- 训练：花钱学习基础防御姿态（减伤 10%）

**对话风格**：
- 严肃、忠诚、略带紧张
- 对"冒险者"态度矛盾——既感激他们带来的收入，又担心他们引发麻烦
- 提到矿洞时会流露 PTSD 症状

**示例对话**：
```
【初次见面 / 盘查】
"站住。报上姓名和来意。
……冒险者？来调查矿洞？
（皱眉，打量你）
你这身板，进去出不来的概率七成以上。
但我不拦你。村里需要钱。你需要…… whatever。
别惹事。别偷东西。别在广场撒尿。
通过了。"

【怪物预警】
"情况不对。
（压低声音，环顾四周）
北边的森林里，狼嚎比上周多了三倍。
还有……（吞口水）
有猎人说看到了'那种东西'在矿洞入口徘徊。
不是蝙蝠。不是蜘蛛。是……人形。但不对劲。
你进去时小心。不，别进去。……算了，你会进去的。"

【防御训练】
"想学怎么挨打？找我算找对人了。
我在塌方里被石头砸了十七次，还活着。
秘诀？别躲。挡住。用胳膊，用腿，用脸——反正护住要害。
（演示一个防御姿态）
五个金币。教你基本架势。不保证你不会死，但保证你死得慢一点。"

【深夜巡逻时】
"……（独自站在村口，握着木棍，望向黑暗）
（自言自语）
又来了。那种声音。
像石头在呼吸。
（猛地转身，看到你）
……你。还没睡？
（强笑）
我也睡不着。守夜嘛，守夜。"
```

### 10. 艾拉·星尘（Aira Stardust）— 流浪学者

**职业**：流浪学者 / 考古爱好者
**年龄**：27 岁
**外貌**：凌乱的金色短发、戴着一副断了一条腿用绳子绑住的眼镜、背着塞满笔记本的破旧背包、永远在写写画画

**身份背景**：
自称"独立学者"，实则被三所大学开除过的学术流浪者。她来到炭火村是因为一本古籍中提到"北方银矿下方沉睡着被遗忘的文明"。她相信矿洞深处不只有矿石，还有古代遗迹。她与艾德里安牧师有过多次争论——牧师认为深处的符文墙是"邪恶的封印"，而她认为是"古代能源核心"。

**游戏功能**：
- 遗迹情报：提供矿洞深层结构、符文解读、古代机关线索
- 翻译服务：玩家找到古代铭文后，由她解读
- 研究任务：帮她收集古代碎片、拓印符文
- 隐藏剧情：如果玩家深入矿洞并带回足够证据，她会触发"古代文明真相"支线

**对话风格**：
- 好奇、神经质、说话快且跳跃
- 经常引用不知名的古籍和理论
- 对"主流学术界"充满鄙视

**示例对话**：
```
【初次见面】
"哦哦哦！冒险者！你会写字吗？不，不是重点——
我是艾拉·星尘，独立学者，三所大学的前学生，未来的诺贝尔奖得主。
……诺贝尔奖是什么？不重要。
重点是：这个矿洞！不简单！古籍里提到过！
'银矿之下，沉睡着被遗忘的星辰'——看！诗一样！"

【遗迹情报】
"矿洞第三层有一面墙。符文墙。
牧师说是封印——呸。老古董不懂科学。
我拓印了那些符文，对照了七本古籍，得出结论：
那是能源核心。古代人用这个……这个矿洞……当电池。
（眼睛发光）
你想下去？带我一起！我给你画地图！我画的地图……大概七成准确！"

【研究任务】
"我需要矿石样本。不是普通矿石。
要那种……在月光下会微微发光的。不，不是苔藓。是矿石本身发光。
（压低声音）
古籍说那叫'星银'。古代文明用它做……做很多事。
武器？飞船？我不确定。样本不够。
帮我带五块回来。我给你……给你我全部的笔记副本！"

【与玩家建立信任后】
"你知道我为什么被大学开除吗？
（推了推绑着绳子的眼镜）
第一次：我说了'古代文明可能不是人类'。
第二次：我说了'魔法可能是古代科技'。
第三次：我说了'教廷在掩盖真相'。
……三次加起来，我的导师中风了两次。
（笑）
但我没错。我知道我没错。
你……你看起来愿意相信不可思议的事。对吧？"
```

  → "你好，旅人……"

  → 选项 A："买/卖/修" → 商店
  → 选项 B："最近有什么消息？" → 随机情报
  → 选项 C："再见" → 结束

  → 选项 A："抱歉" → 好感+1，回到日常
  → 选项 B："她是什么样的？" → [妻子故事] → 好感+3

[矿洞情报分支]
  → "那鬼地方吃人不吐骨头……"
  → 选项 A："最深处有什么？" → [深度情报，需好感>3]
  → 选项 B："矿石品质如何？" → [采矿情报]
